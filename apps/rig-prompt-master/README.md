# RIG Prompt Master Desktop Launcher

This folder owns the single Desktop launcher for the repo-hosted RIG Prompt Master app.

- App bundle: `RIG Prompt Master.app`
- Backup launcher: `RIG Prompt Master.command`
- Local URL: `http://127.0.0.1:8767`

The launcher starts `python/rig/rig_app_server.py` from this repository and opens the local web app. It replaces the old standalone 4188 command-surface app; the command catalog is now served inside RIG Prompt Master.
